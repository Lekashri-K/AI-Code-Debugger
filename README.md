# 🐍 AI Code Debugger

An AI-powered web app that helps beginners debug Python code by explaining errors in simple language and suggesting fixes.

---

## 🚀 Features
- AI-based Python error explanation
- Simple breakdown: What went wrong, why, and how to fix it
- Built-in code execution sandbox
- Beginner-friendly debugging assistant

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- Google Generative AI (Gemini)
